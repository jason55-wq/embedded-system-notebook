# Arduino 筆記與 Code 機器人

這是一個可直接在本機打開的前端小工具，適合拿來整理 Arduino 學習筆記、儲存模組紀錄，並快速產生常見專案的範例程式。

## 功能

- 本機筆記儲存
- 常見 Arduino 程式碼產生器
- 簡易除錯建議助手
- 純 HTML / CSS / JavaScript，不需要先安裝套件

## 使用方式

1. 可以直接用瀏覽器打開 `index.html`
2. 或在此資料夾執行 `python server.py`，再打開終端機顯示的本機網址
3. 在筆記區輸入模組名稱、標籤與內容後按下「儲存筆記」
4. 在 Code Builder 選擇專案類型後取得範例程式
5. 在除錯助理輸入問題描述，取得排查方向
